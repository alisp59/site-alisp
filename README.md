# Site web de l'ALISP

[alisp.fr](https://alisp.fr)

Le site est généré avec [Quarto](https://quarto.org/) et hébergé sur [Netlify](https://www.netlify.com/).

Pour alimenter le site, tout se fait via une interface visuelle, dans une branche à part soumise à validation (impossible de "casser" le site)

1. Aller sur [alisp.fr/admin](https://alisp.fr/admin)
2. Se connecter avec le compte GitHub alisp59
3. Rédiger dans l'éditeur visuel
5. Cliquer sur **Save** puis **Set status > In Review**
6. Je reçois une notification, valide puis update du site (automatisable mais dans un second temps)
